Take note

Use '()' for 2 or more digit exponents.
x^2a = x^2a
x^(2a) = x^2^a

Use '()' for grouping.
4x - 2 / 2x + 8
= 4x -\frac{2}{2x} + 8
(4x - 2) / (2x + 8)
= \frac{4x - 2}{2x + 8}

Use 'sqrt()' for square root.
5x + sqrt(x) = 5x + \sqrt x

Use 'Abs()' for square root (uppercase 'A').
9 / Abs(-3) = \frac{9}{|-3|}
